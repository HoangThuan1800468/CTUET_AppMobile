# CTUET_AppMobile
 - Phần mềm nền tảng điện thoại (Android) ngôn ngữ Java
 - Database: Sqlite
 - Chức năng:
   * Sinh viên:
     + Quản lý thông tin cá nhân
     + Xem lịch học
     + Xem điểm 
     + Nhận thông báo từ giảng viên
   * Giảng viên:
     + Xem lịch giảng dạy
     + Nhập điểm 
     + Gửi thông báo về lớp học cho sinh viên
   * Quản trị viên:
     + Quản lý lớp học phần
     + Quản lý tài khoản
